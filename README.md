# 🚀 BDH Linux IDE 

**BDH Linux IDE** is a simple, lightweight, and fast terminal-based Integrated Development Environment (IDE). Built for hardcore developers, it seamlessly integrates **Tmux**, **Ranger**, and **Nano** into a single 3-pane workspace.

---

## ✨ Features

- **3-Pane Layout**: A perfectly split view featuring:
  - 📂 **Ranger**: File manager for quick navigation.
  - 📝 **Nano**: Text editor for distraction-free coding.
  - 💻 **Terminal**: Interactive shell for running commands.
- **Auto-Syncing Workspace**: Hitting Enter on a file in Ranger automatically opens it in Nano and changes your bottom Terminal's directory to match!
- **Pure Terminal Experience**: Zero GUI overhead, fully keyboard-driven.

---

## 📥 Installation

Clone the repository and run the installation script:

```bash
git clone [https://github.com/your-username/bdh-linux-ide.git](https://github.com/your-username/bdh-linux-ide.git)
cd bdh-linux-ide
sudo ./install.sh


<h1>🚀 Usage</h1>
​After installation, you can launch or close the IDE from anywhere in your terminal:
<pre>
​Start IDE: bdh-ide
​Kill all sessions: bdh-ide-kill
​⌨️ Shortcuts
​Alt + Arrow Keys: Navigate between Ranger, Nano, and Terminal panes.
​F4: Force close the IDE session safely.

</pre>
